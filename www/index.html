Tech Stack

🔧 Build

Gradle 9.6 (nightly) with AGP 9.1, targeting Android 16 (API 36)

💜 UI

Jetpack Compose with Material 3. Custom animation system for glitch effects and timeline interactions.

💾 Persistence

Room Database with reactive StateFlow for profile management and game state. Zero-setup SQLite.

⚙️ Engine

Pure Kotlin tick system in headless `:core` module. No Android framework in domain logic.

🗺️ Navigation

Navigation Compose with type-safe route encoding. Account → Game → Skill/Bank/Combat flows.

🚀 Future

C++ OpenGL island for high-performance rendering. Architecture is additive — no breaking changes planned.

Design Philosophy

Keep It Simple. No premature abstractions. You Aren't Gonna Need It. No speculative features. Don't Repeat Yourself. One source of truth for rules and state. Fail Fast. Errors surface immediately; no silent failures.

Every line of code earns its place. Animations are performant Canvas operations, not layout thrashing. Domain logic is pure and testable. UI is a thin reactive layer over immutable state.
