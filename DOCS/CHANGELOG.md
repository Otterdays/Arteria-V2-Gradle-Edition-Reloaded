# Changelog
All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [V2.8.0] - 2026-04-21
### Added
- **Forging Skill**: Craft equipment from Smithing bars.
- **Jewelcrafting Skill**: Cut gems and craft rings/amulets (Mining gems loop).
- **Firemaking Skill**: Burn logs for ashes (future Alchemy prep).
- **Mining Expansion**: Added Gem Rocks (Sapphire, Emerald, Ruby, Diamond).
- **Holographic UI (CyberHUD)**: Added moving grids and scanlines for "High-Juice" feel.
- **Visual Polish**: Circular "Energy Pulse" for training, glassmorphism card highlights.
- **ArteriaPalette Expansion**: New "HoloBlue", "CyberPink", "MatrixGreen" tokens in `ArteriaTheme.kt`.

## [V2.7.0] - Prior
- Base Gradle/Kotlin architecture.
- Core 12 skills (Mining, Woodcutting, Smithing, etc.).
- Tick Engine and offline progress.
